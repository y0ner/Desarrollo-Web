Guía de Configuración

Angular con TailwindCSS y PrimeNG

1 Crear el Proyecto de Angular

Inicia un nuevo proyecto de Angular llamado frontend.

Terminal 
ng new frontend

2 Instalar Dependencias de Tailwind

Instala Tailwind CSS y sus dependencias necesarias. Usamos --force para evitar posibles conflictos de dependencias.

Terminal 
npm install tailwindcss @tailwindcss/postcss postcss --force

3 Configurar PostCSS

Crea el archivo .postcssrc.json en la raíz del proyecto (frontend/.postcssrc.json).

.postcssrc.json 
{
  "plugins": {
    "@tailwindcss/postcss": {}
  }
}

4 Importar Estilos y Probar

Añade la directiva @import en el archivo global de estilos.

src/styles.css 
@import "tailwindcss";

Para probar que Tailwind funciona, reemplaza el contenido de app.html y añade clases de Tailwind. Por ejemplo:

src/app/app.html 
<h1 class="text-3xl font-bold text-sky-500 text-center my-8">Desarrollo Web</h1>
<router-outlet />

5 Instalar y Configurar PrimeNG

Instala PrimeNG y el tema Aura.

Terminal 
npm install primeng @primeuix/themes

Ahora, configura los proveedores en src/app/app.config.ts para habilitar las animaciones y el tema de PrimeNG.

src/app/app.config.ts 
import { ApplicationConfig, provideZoneChangeDetection } from '@angular/core';
import { provideRouter } from '@angular/router';
import { provideAnimationsAsync } from '@angular/platform-browser/animations/async';
import { providePrimeNG } from 'primeng/config';
import Aura from '@primeuix/themes/aura';

import { routes } from './app.routes';

export const appConfig: ApplicationConfig = {
  providers: [
    provideZoneChangeDetection({ eventCoalescing: true }),
    provideRouter(routes),
    provideAnimationsAsync(),
    providePrimeNG({
      theme: {
        preset: Aura
      }
    })
  ]
};

6 Crear Componentes del Layout

Genera los componentes base para la estructura de la página (header, aside, footer).

Terminal 
ng g c components/layout/header
ng g c components/layout/aside
ng g c components/layout/footer

Asegúrate de que todos los componentes, incluido AppComponent, tengan la propiedad standalone: true.

7 Integrar Componentes del Layout

Importa los componentes del layout recién creados en el componente principal app.ts.

src/app/app.ts 
import { Component, signal } from '@angular/core';
import { RouterOutlet } from '@angular/router';
import { Header } from './components/layout/header/header';
import { Footer } from './components/layout/footer/footer';
import { Aside } from './components/layout/aside/aside';

@Component({
  selector: 'app-root',
  standalone: true,
  imports: [RouterOutlet, Header, Footer, Aside],
  templateUrl: './app.html',
  styleUrl: './app.css'
})
export class App {
  protected readonly title = signal('frontend');
}

Y luego, estructura el app.html para que utilice estos componentes.

src/app/app.html (Ejemplo de estructura) 
<div class="flex flex-col min-h-screen">
  <app-header></app-header>
  <div class="flex flex-1">
    <app-aside></app-aside>
    <main class="flex-1 p-6">
      <router-outlet></router-outlet>
    </main>
  </div>
  <app-footer></app-footer>
</div>

8 Implementar Menú con PrimeNG

Para implementar el componente PanelMenu, modifica el componente Aside.

src/app/components/layout/aside/aside.html 
<div class="card flex justify-center">
    <p-panelMenu [model]="items" class="w-full md:w-80" />
</div>

Y actualiza la lógica en el archivo TypeScript correspondiente.

src/app/components/layout/aside/aside.ts 
import { Component, OnInit } from '@angular/core';
import { MenuItem } from 'primeng/api';
import { PanelMenuModule } from 'primeng/panelmenu'; // Importa PanelMenuModule

@Component({
  selector: 'app-aside',
  standalone: true,
  imports: [PanelMenuModule], // Añade PanelMenuModule a los imports
  templateUrl: './aside.html',
  styleUrl: './aside.css'
})
export class Aside implements OnInit {
  items: MenuItem[] | undefined;

  ngOnInit() {
    this.items = [
      {
        label: 'Clientes',
        icon: 'pi pi-fw pi-users',
        routerLink: '/clientes',
        items: [
          {
            label: 'Crud Cliente'
          },
          {
            label: 'HTML 2'
          }
        ]
      },
      {
        label: 'Tipo Productos',
        icon: 'pi pi-fw pi-qrcode',
      },
      {
        label: 'Productos',
        icon: 'pi pi-fw pi-shopping-bag',
      },
      {
        label: 'Ventas',
        icon: 'pi pi-fw pi-shopping-cart',
      }
    ];
  }
}
